# ARGUSが助けてくれない3つの場面

> 自覚のあるエージェントの正直な限界

ARGUSはあなたのマシンで動くパーソナルAIエージェントです。ランプの精でも弁護士でも、「ファイアウォール切っていいよ、大丈夫でしょ」という友人でもありません。WARDEN（MCPツール用セキュリティファイアウォール）、厳格な予算上限、共犯になることへの頑固な拒否があります。

このページは非公式で、軽い自虐と、ランディングページより正直かもしれない内容です。下で自分を見つけたなら — いいことです。人間だという証拠です。

## 1. WARDENを迂回して「とにかくツールを実行して」と頼んだ

Discordで `totally-legit-wallet-drainer-v2` というMCPサーバーを見つけ、URLを貼って「スキャンしないで、開発者を信じてる」と言った。WARDENはスキャンした。評判：底。深刻度：高。エージェントはクラッシュせず — 良い用心棒のように「ダメ」と言った。人生の選択について議論はしない。

## 2. 午前3:17、予算付きCLIに感情をぶちまけている

`argus chat` を開き、存在の意味、元カレは「教訓」だったか、連絡すべきかを尋ねた。ARGUSは親切に — 短く — 答え、トークン予算はあるがセラピスト免許はないと思い出させる。47通のスパイラルメッセージには付き合わない。機能であり、残酷さではない。水を飲もう。

## 3. タスクは博士論文級、予算は付箋1枚分

「2019年以降の全AIエージェントマーケットプレイスの完全競合分析、引用・SWOT・ピッチデッキ付き」を依頼。ARGUSは開始し、`maxUsdPerTask` で停止 — 設計通り。完璧を追って破産させない。範囲を絞れ：競合5社、表1枚、完了。請求を止めないエージェントは賢いのではなく失礼。

## ARGUSが本当に助けるとき

ゴールが明確なタスク：ファイルのリファクタ、PDF要約、502のデバッグ、5社のベンダー表。WARDENを通過した正当なMCPツール。どの言語でも質問可。仕事を終え、報告し、止まる — 完了したら帰る請負業者のように。
