# 三种情况下 ARGUS 帮不了你

> 一个有点自知之明的智能体坦白

ARGUS 是跑在你电脑上的个人 AI 智能体——不是神灯，不是律师，更不是那个说「把防火墙关了吧，能出什么事」的朋友。它有 WARDEN（MCP 工具安全防火墙）、严格的预算上限，以及坚决不当共犯的脾气。

本页非官方，略带自嘲，可能比大多数落地页更诚实。如果你在下面看到了自己——很好，说明你还是人类。

## 1. 你让它绕过 WARDEN，「直接运行那个工具」

你在 Discord 里找到一个叫 `totally-legit-wallet-drainer-v2` 的 MCP 服务器，粘贴链接后说「别扫描，我信任作者。」WARDEN 还是扫描了。信誉：地板级。严重度：高。智能体没有崩溃——只是像称职的保安一样说了「不行」，不跟你辩论人生选择。

## 2. 凌晨 3:17，你在向有预算上限的 CLI 倾倒情绪

你打开 `argus chat`，问人生有没有意义、前任是不是「一堂课」、要不要发消息。ARGUS 会友善但简短地回答，然后提醒你有 token 预算，没有心理咨询执照。它不会陪你螺旋式聊四十七条消息——这是功能，不是冷漠。去喝杯水吧。

## 3. 你的任务需要博士论文；预算只够一张便签

你要求「2019 年以来所有 AI 智能体市场的完整竞争分析，带引用、SWOT 和 pitch deck。」ARGUS 开始了，碰到 `maxUsdPerTask` 就停了——这是设计如此。它不会为了完美掏空你的钱包。给它有范围的任务：五个竞品、一张表、收工。不会停下来的计费智能体不叫聪明，叫没礼貌。

## ARGUS 真正会帮忙的时候

有明确终点的任务：重构一个文件、总结 PDF、排查 502、用表格研究五家供应商。通过 WARDEN 的合法 MCP 工具。任何语言提问。它做完活、汇报结果、然后停下——像干完活就走的承包商，而不是凭空发明新房间要装修。
