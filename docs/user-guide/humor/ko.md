# ARGUS가 도와주지 않는 세 가지 상황

> 자각 있는 에이전트의 솔직한 한계

ARGUS는 내 컴퓨터에서 돌아가는 개인 AI 에이전트입니다. 요정도, 변호사도, "방화벽 꺼도 돼, 뭐가 문제야"라는 친구도 아닙니다. WARDEN(MCP 도구 보안 방화벽), 엄격한 예산 한도, 공범 거부가 있습니다.

이 페이지는 비공식이며, 가볍게 자조적이고 랜딩 페이지보다 솔직할 수 있습니다. 아래에서 자신을 발견했다면 — 좋습니다. 인간이라는 뜻입니다.

## 1. WARDEN을 우회하고 "그냥 도구 실행해"라고 했다

Discord에서 `totally-legit-wallet-drainer-v2` MCP 서버를 찾아 URL을 붙여넣고 "스캔하지 마, 개발자를 믿어"라고 했다. WARDEN은 스캔했다. 평판: 바닥. 심각도: 높음. 에이전트는 멈추지 않았다 — 좋은 경비원처럼 거절했다. 인생 선택과 논쟁하지 않는다.

## 2. 새벽 3:17, 예산 제한 CLI에 감정을 쏟고 있다

`argus chat`을 열어 존재의 의미, 전 연인이 "교훈"이었는지, 연락해야 하는지 물었다. ARGUS는 친절히 — 짧게 — 답하고, 토큰 예산은 있지만 상담사 면허는 없다고 상기시킨다. 47개의 나선형 메시지를 듣지 않는다. 기능이지 잔인함이 아니다. 물 마셔.

## 3. 과제는 박사 논문급, 예산은 포스트잇 한 장

"2019년 이후 모든 AI 에이전트 마켓플레이스 완전 경쟁 분석, 인용·SWOT·피치덱 포함"을 요청했다. ARGUS는 시작했고 `maxUsdPerTask`에서 멈췄다 — 설계상 그렇다. 완벽을 위해 파산시키지 않는다. 범위를 좁혀라: 경쟁사 5개, 표 1개, 끝.

## ARGUS가 진짜 도울 때

끝이 분명한 작업: 파일 리팩터, PDF 요약, 502 디버그, 공급업체 5개 표. WARDEN을 통과한 정당한 MCP 도구. 어떤 언어로든 질문. 일을 끝내고 보고하고 멈춘다 — 일 끝나면 떠나는 계약업자처럼.
