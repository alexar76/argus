# Tri situácie, keď ARGUS nepomôže

> Úprimné limity od agenta so sebaióniou

ARGUS je osobný AI agent na vašom počítači — nie džin, nie advokát a určite nie kamarát, čo hovorí «vypni firewall, čo sa môže stať?». Má WARDEN (bezpečnostnú bránu pre MCP nástroje), tvrdé rozpočtové limity a tvrdohlavé odmietanie byť spolupáchateľom.

Táto stránka je neoficiálna, mierne sebaióničná a pravdepodobne úprimnejšia ako väčšina landingov. Ak sa nižšie spoznáte — dobre. Ste človek.

## 1. Požiadali ste o obídenie WARDEN a «proste spusti nástroj»

Našli ste MCP server na Discorde `totally-legit-wallet-drainer-v2`. Vložili URL a povedali «neskenuj, verím devovi». WARDEN aj tak naskenoval. Reputácia: dno. Závažnosť: vysoká. Agent nespadol — povedal nie, ako dobrý vyhadzovač.

## 2. Je 3:17 v noci a vylievate emócie do CLI s rozpočtom

Otvorili ste `argus chat` a pýtate sa na zmysel života, či bol ex «lekciou» a či písať. ARGUS odpovie zdvorilo — stručne — a pripomenie rozpočet tokenov, nie terapeutickú licenciu. Štyridsaťsedem špirálových správ nepočúva. Je to funkcia, nie krutosť. Vypite vodu.

## 3. Úloha chce dizertáciu; rozpočet kúpi post-it

Chceli ste «úplnú konkurenčnú analýzu všetkých AI marketplace od 2019 s citáciami, SWOT a pitch deckom». ARGUS začal, narazil na `maxUsdPerTask` a zastal — zámerne. Nerozvráti vás v honbe za dokonalosťou. Dajte scoped úlohu: päť konkurentov, jedna tabuľka, hotovo.

## Keď ARGUS naozaj pomáha

Jasné úlohy s cieľom: refaktor súboru, zhrnutie PDF, debug 502, tabuľka piatich dodávateľov. Legitímne MCP nástroje cez WARDEN. Otázky v akomkoľvek jazyku. Dokončí, nahlási, zastaví — ako dodávateľ, čo odíde po práci.
