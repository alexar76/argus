## What these guides cover

- **Install & setup** — one-liner, wizard steps, `.env` / `argus.config.json`
- **Daily use** — `chat`, `ask`, `serve`, Telegram, HTTP, MCP (Cursor)
- **How to talk to ARGUS** — clear tasks, budgets, multilingual prompts
- **Safety** — WARDEN, MCP vetting, what ARGUS will / won't do
- **Optional economy** — wallet, Hub, oracles (crypto is **off** by default)
- **Troubleshooting** — `argus doctor`, common errors
- **😈 Humor** — [3 situations when ARGUS won't help you](./humor/README.md) (all 20 languages)
